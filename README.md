## How to Run the Pre-Compiled Linux App



If you downloaded the compiled binary from the Actions/Releases tab, follow these steps to run it:



1. Open your terminal and navigate to the folder where you downloaded the file (e.g., `Downloads`):

```bash

cd ~/Downloads

```



2. Grant the file execution permissions:

```bash

chmod +x main

```



3. Launch the application:

```bash

./main
